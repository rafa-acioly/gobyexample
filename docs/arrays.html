Go por Exemplo: Arrays

Em Go, um array é uma sequência numerada de elementos de um tamanho especifico.

 
package main

import "fmt"

func main() {

Aqui criamos um array a que conterá exatamente 5 inteiros. O tipo dos elementos e o tamanho do array são ambos parte do tipo do array. Por padrão um array é zero-valued, que para inteiros significa zeros.

 
    var a [5]int
    fmt.Println("emp:", a)

Nós podemos definir um valor para um determinado indice usando a sintaxe array[index] = value e para pegar o valor é usado array[index]

 
    a[4] = 100
    fmt.Println("set:", a)
    fmt.Println("get:", a[4])

A função len já vem embutida junto ao Go e retorna o tamanho de um array.

 
    fmt.Println("len:", len(a))

Use esta sintaxe para declarar e inicializar um array com valores usando apenas uma linha.

 
    b := [5]int{1, 2, 3, 4, 5}
    fmt.Println("dcl:", b)

O tipo array é uni-dimensional, mas você pode compor mais tipos para criar um array multi-dimensional

 
    var twoD [2][3]int
    for i := 0; i < 2; i++ {
        for j := 0; j < 3; j++ {
            twoD[i][j] = i + j
        }
    }
    fmt.Println("2d: ", twoD)
}

Note que array aparecem na forma de [v1 v2 v3 ...] quando exibido com fmt.Println.

 
$ go run arrays.go
emp: [0 0 0 0 0]
set: [0 0 0 0 100]
get: 100
len: 5
dcl: [1 2 3 4 5]
2d:  [[0 1 2] [1 2 3]]
