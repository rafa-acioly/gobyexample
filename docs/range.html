Go por Exemplo: Range

range itera sobre elementos em diversos tipos de estruturas de dados. Vamos ver como usar range com algumas estruturas de dados que já aprendemos.

Aqui nós usmamos range para somar os numeros em um slice. Arrays funcionam bem desta maneira também.

 
    nums := []int{2, 3, 4}
    sum := 0
    for _, num := range nums {
        sum += num
    }
    fmt.Println("sum:", sum)

range com arrays e slices provêem ambos, valores e indices para cada entrada/par. Acima nós não precisavamos do indice, então o ignoramos com um identificados vazio _. Porém algumas vezes nós realmente precisamos do indice.

 
    for i, num := range nums {
        if num == 3 {
            fmt.Println("index:", i)
        }
    }

range em map itera sobre o par de chaves/valores.

 
    kvs := map[string]string{"a": "apple", "b": "banana"}
    for k, v := range kvs {
        fmt.Printf("%s -> %s\n", k, v)
    }

range também pode iterar apenas sobre as chaves de um map.

 
    for k := range kvs {
        fmt.Println("key:", k)
    }

range em strings itera sobre o ponto de código Unicode. O primeiro valor é o bite inicial e o segundo é ele próprio.

 
    for i, c := range "go" {
        fmt.Println(i, c)
    }
}
