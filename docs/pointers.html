Go por Exemplo: Ponteiros

Go suporta ponteiros, permitindo que você passe referências para valores e registros em seu programa.

 
package main

import "fmt"

Iremos mostrar como ponteiros funcionando em contraste com valores em 2 funções. zeroval e zeroptr. zeroval tem um parâmetro int, então o argumento será passado por valor. zeroval receberá uma copia distinta de ival através de quem chamar a função.

 
func zeroval(ival int) {
    ival = 0
}

zeroptr possui um parâmetro *int, significando que recebe um parâmetro ponteiro int. O código *iptr no corpo da função desreferencia o ponteiro no endereço de memória para o valor atual no endereço. Atribuindo o valor para o ponteiro deferenciado muda o valor do endereço referenciado.

 
func zeroptr(iptr *int) {
    *iptr = 0
}

func main() {
    i := 1
    fmt.Println("initial:", i)

    zeroval(i)
    fmt.Println("zeroval:", i)

A sintaxe &i é o endereço na memória de i, i.e um ponteiro para i.

 
    zeroptr(&i)
    fmt.Println("zeroptr:", i)

Ponteiros podem também ser exibidos.

 
    fmt.Println("pointer:", &i)
}

zeroval não altera o i no main, mas zeroptr alterado porque ele possui uma referencia para o endereço de memória na variável.

 
$ go run pointers.go
initial: 1
zeroval: 1
zeroptr: 0
pointer: 0x42131100
