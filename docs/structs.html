Go por Exemplo: Estruturas (structs)

structs m Go são um tipo de coleção de campos. Muito util para agrupar dados para formar registros.

 
package main

import "fmt"

Esta estrutura person contêm os campos name e age.

 
type person struct {
    name string
    age  int
}

func main() {

Esta sintaxe cria uma nova estrutura.

 
    fmt.Println(person{"Bob", 20})

Você pode nomear os campos quando inicializar uma estrutura.

 
    fmt.Println(person{name: "Alice", age: 30})

Campos omitidos serão zero-valued.

 
    fmt.Println(person{name: "Fred"})

O prefixo & define um ponteiro para a estrutura.

 
    fmt.Println(&person{name: "Ann", age: 40})

Acesso campos de estruturas com um ponto.

 
    s := person{name: "Sean", age: 50}
    fmt.Println(s.name)

Você também pode usar pontos com ponteiros de estrutura - o ponteiro é automáticamente desreferenciado.

 
    sp := &s
    fmt.Println(sp.age)

Estruturas são mutáveis.

 
    sp.age = 51
    fmt.Println(sp.age)
}
