Go por Exemplo: If/Else

Ramificações com if e else utiliando Go são diretas

 
package main

import "fmt"

func main() {

Aqui está um exemplo básico.

 
    if 7%2 == 0 {
        fmt.Println("7 is even")
    } else {
        fmt.Println("7 is odd")
    }

Você pode ter uma declaração de if sem utilizar o else.

 
    if 8%4 == 0 {
        fmt.Println("8 is divisible by 4")
    }

Uma declaração pode proceder de uma condição; qualquer variável declarada dentro de um if ficará disponivel em todas as declarações de else/else if subsequentes.

 
    if num := 9; num < 0 {
        fmt.Println(num, "is negative")
    } else if num < 10 {
        fmt.Println(num, "has 1 digit")
    } else {
        fmt.Println(num, "has multiple digits")
    }
}

Perceba que em Go, você não precisa utilizar parênteses ao redor as condições, mas as chaves são obrigatórias.

 
$ go run if-else.go 
7 is odd
8 is divisible by 4
9 has 1 digit

Em Go não existem operadores ternários, logo, você precisará utilizar uma declaração de if completa até para condições mais simples.
