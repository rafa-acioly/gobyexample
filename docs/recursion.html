Go por Exemplo: Recursão

Go suporte funções recursivas. Aqui está um exemplo classico de uma função factorial

 
package main

import "fmt"

Esta função fact chama a sí própria até alcançar a condição de fact(0).

 
func fact(n int) int {
    if n == 0 {
        return 1
    }
    return n * fact(n-1)
}

func main() {
    fmt.Println(fact(7))
}

$ go run recursion.go
