Go por Exemplo: Variáveis

Em Go, variáveis são explicitamente declaradas e usadas pelo compilador.

 
package main

import "fmt"

func main() {

var declara 1 ou mais variáveis.

 
    var a string = "initial"
    fmt.Println(a)

Você pode declarar múltiplas variáveis de uma vez.

 
    var b, c int = 1, 2
    fmt.Println(b, c)

Go irá inferir o tipo da variável inicializada.

 
    var d = true
    fmt.Println(d)

Variables declared without a corresponding initialization are zero-valued. For example, the zero value for an int is 0.

Variáveis declaradas sem a inicialização são zero-valued. Por exemplo, o valor zero para um int é 0

 
    var e int
    fmt.Println(e)

A sintaxe := é um encurtador para declarar uma inicialização de variável, o que corresponderia á var f string = "short"

 
    f := "short"
    fmt.Println(f)
}

$ go run variables.go
