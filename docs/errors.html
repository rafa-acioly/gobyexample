Go por Exemplo: Erros

Em Go é idiomático comunicar erros através de um valor explícito e separado em retornos. Isto constrata com as exceções usadas em linguagens como Java e Ruby e o valor/resultado isolado de sobrecarrregamento em C.

 
package main

import "errors"
import "fmt"

Por convenção, erros são o último valor retornado e possuem o tipo error, uma interface já embutida na linguagem.

 
func f1(arg int) (int, error) {
    if arg == 42 {

errors.New constrói um error básico com a mensagem de erro definida.

 
        return -1, errors.New("can't work with 42")

    }

Um valor nil definido na posição de erro indica que não houve erro.

 
    return arg + 3, nil
}

É possível usar tipos custumizados como erros apenas implementando o método Error(). Aqui está um variante no exemplo acima que usa um tipo personalizado para representar explicitamente um erro de argumento.

 
type argError struct {
    arg  int
    prob string
}

func (e *argError) Error() string {
    return fmt.Sprintf("%d - %s", e.arg, e.prob)
}

func f2(arg int) (int, error) {
    if arg == 42 {

Neste caso nós usamos a sintaxe &argError para construir uma nova struct, fornecendo valores para os dois campos arg e prob.

 
        return -1, &argError{arg, "can't work with it"}
    }
    return arg + 3, nil
}

Os dois loops abaixo testam cada retorno de erro de nossas funções. Note que usamos a verificação do erro em uma única linha (inline) no if isto é muito comum em códigos Go.

 
    for _, i := range []int{7, 42} {
        if r, e := f1(i); e != nil {
            fmt.Println("f1 failed:", e)
        } else {
            fmt.Println("f1 worked:", r)
        }
    }
    for _, i := range []int{7, 42} {
        if r, e := f2(i); e != nil {
            fmt.Println("f2 failed:", e)
        } else {
            fmt.Println("f2 worked:", r)
        }
    }

Se você quiser usar os dados de um erro customizado, você precisará pegar o erro como uma instância do erro customizado através de asserção de tipos.

 
    _, e := f2(42)
    if ae, ok := e.(*argError); ok {
        fmt.Println(ae.arg)
        fmt.Println(ae.prob)
    }
}
