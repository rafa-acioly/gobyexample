Go por Exemplo: Channels

Canais são os pipes que conectam as goroutines. Você pode enviar valores para os canais a partir de uma goroutine e receber estes mesmos valores em outra goroutine.

 
package main
            
import "fmt"
            
func main() {

Crie um novo canal com make(chan val-type) Canais são tipados pelos valores que transitam neles.

 
    messages := make(chan string)

Envie um valor para um canal usando a sintaxe channel <-. Aqui nós enviamos "ping" para o canal de messages que criamos acima a partir de uma nova goroutine.

 
    go func() { messages <- "ping" }()

A sintaxe <-channel recebe um valor de um canal. Aqui nós recebemos a mensagem "ping" que nós enviamos acima e exibimos ela.

 
    msg := <-messages
                    fmt.Println(msg)

When we run the program the "ping" message is successfully passed from one goroutine to another via our channel.

 
$ go run channels.go 
                      ping

Por padrão enviar e receber valores através dos canais bloqueia ambos remetente e recebedor até que eles estejam prontos. Esta propriedade nos permite esperar no final do programa a mensagem "ping" sem termos que usar nenhuma outra sincronização.
