Go por Exemplo: Closures

Go suporte funções anônimas, na qual produzem closures. Funções anônimas são úteis quando você quer definir uma função sem precisar nomea-la.

 
package main

import "fmt"

Esta função intSeq retorna outra função, na qual definimos anônimamente no corpo de intSeq. O retorno da função encapsula sobre a variável i para formar um closure.

 
func intSeq() func() int {
    i := 0
    return func() int {
        i += 1
        return i
    }
}

O valor desta função captura o próprio valor de i, que atualizara cada vez que chamar-mos nextInt.

 
    nextInt := intSeq()

Veja o efeito da closure chamando nextInt algumas vezes.

 
    fmt.Println(nextInt())
    fmt.Println(nextInt())
    fmt.Println(nextInt())

Para confirmar que o estado é único para aquela função em particular, crie e teste uma nova variável.

 
    newInts := intSeq()
    fmt.Println(newInts())
}
